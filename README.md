## About Me
I am a beginner in Cyber Security, focusing on network and web security. I practice in safe lab environments such as TryHackMe and OWASP Juice Shop. This portfolio showcases my exercises, tool usage, mini-projects, and earned certificates.

## Skills
- Networking: IP, ports, DNS, protocols
- Linux commands: ls, cd, mkdir, cat, chmod
- Security tools: Nmap, Wireshark, Nikto, Hydra
- Web security: XSS, SQL Injection
- Log analysis and traffic monitoring

## Portfolio Structure
- **Network-Security** - Exercises on networks, including Nmap scans, Wireshark captures, TCP handshake, DNS/HTTP analysis.
- **Web-Security** - Exercises on web security, like OWASP Juice Shop, XSS, SQLi, with mitigation notes and screenshots.
- **Tools** - Notes on security tools with commands, outputs, and analysis.
- **TryHackMe** - Summary of completed beginner rooms with key learnings.
- **Mini-Project** - Small practical projects like Linux firewall setup or honeypot experiments.
- **Certificate** - Earned badges and certificates in PDF or image format.
